# Historical Planning Artifacts

> **Status: Historical.** These documents capture the multi-agent design and
> decision process that produced ExecutionKit v0.1. They are **not** active specs.
>
> For current architecture and design rationale, see:
> - [`docs/architecture.md`](../architecture.md) — module map, layering, data flow
> - [`docs/adr/`](../adr/) — formal Architecture Decision Records (when added)
> - [`CONTRIBUTING.md`](../../CONTRIBUTING.md) — active development guidelines
>
> The decisions that survived the planning process are extracted into ADRs.
> The files here serve as primary sources for that extraction — not as guidance
> for contributors.
